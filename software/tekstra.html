Tekstra

An advanced and extensive text editor/notepad!

Tekstra is a free application designed for more extensive editing of text (including .rtf files) than the default MS Notepad. The program is aimed toward intermediate to advanced users who require greater control and flexibility in text formatting (though beginners may still find use in the software). With Tekstra, you can edit both plain text files and RTF (Rich Text Format) documents, with a range of features including customizable fonts, paragraph styling, and other basic formatting tools as well as more advanced "power tools" for heavy or repetitive work. Whether you're writing down some quick notes, managing data logs, or working with formatted documents, Tekstra offers a lightweight yet capable environment for efficient text editing.

Features

Layout and View

  • 2 Plain Text Viewing Fonts

    - Allows viewing plain text in a proportional or monospace font.

  • Stay on Top

    - Forces the application to stay on top of other windows.

  • Last Window Placement Memory

    - Can remember the position of the window and automatically load to that location.
Basic Functionality

  • Find and Replace

    - Generic find and replace functionality.

  • RTF Formatting

    - Allows for formatting text with bold, italic, fonts, colors, etc.

  • Controlled Exporting and Importing

    - Allows you to chose what encoding to use when exporting/importing.

  • Document File Renaming

    - Allows you to rename the current file without closing the application.

  • Recent and Pinned Files

    - Remembers recently opened files and allows you to pin files for quick access later.
Power Tools (Advanced Functions)

  • Remove Lines With

    - Removes all lines which contain the specified text.

  • Remove Duplicate Lines

    - Removes all but the first occurrence of any lines which are identical to other lines.

  • Remove Empty Lines

    - Removes all empty/blank lines.

  • Add to Each Line

    - Adds the specified text to the beginning or end of each line.

  • Remove from Each Line

    - Removes the specified section of text from each line.

  • Split

    - Splits the text into lines.

  • Make Single Line

    - Combines all lines into one line with optional separator.
